# 昨夜寝るのが遅くなって、今朝二度寝しそうになったけど起きられた

I ended up staying up late last night, and this morning I almost fell back asleep — but I managed to get up.
I'm actually proud of myself.

I stayed up quite late last night and nearly hit the snooze button this morning, but I managed to get myself up.
I can feel myself growing.

I was up way too late last night, and this morning I was so close to going back to sleep — but nope, I dragged myself up.
Growth, baby.


# 帰り際の一言：研究室の友人への声かけ

Alright, I'm heading home!
I'm going to head home now.
Okay, I'm out!
See ya!


# ここがポイントです。
# "I'm going to home" は誤り。home は副詞なので to は不要です。
# "go home" / "head home" がセットで正しい形です。

I'm heading home.
I'm going to head home now.


# 関連表現まとめ

I'm heading home.
I'm out!
Alright, I'm out!
See ya!
I'll be heading out for the day.
Okay, I'm out! See ya!
